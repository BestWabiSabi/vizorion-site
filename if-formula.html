IF Formula

=IF(logical_test, value_if_true, value_if_false)

The IF formula lets you test a condition and return different values depending on whether the condition is TRUE or FALSE.
Example:
=IF(A1>10, "Over 10", "10 or less")
This checks if the value in A1 is greater than 10. If yes, it returns “Over 10”, otherwise “10 or less”.
