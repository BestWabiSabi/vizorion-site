VLOOKUP Formula

The VLOOKUP function allows you to search for a value in the first column of a range or table and return a value in the same row from another column.

  • Syntax: =VLOOKUP(lookup_value, table_array, col_index_num, [range_lookup])
  • Example: =VLOOKUP("Apple", A2:C10, 3, FALSE)
    This finds "Apple" in the first column of A2:C10 and returns the value from the 3rd column of that row.
    Use FALSE for an exact match.
