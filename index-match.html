INDEX & MATCH Formula

=INDEX(return_range, MATCH(lookup_value, lookup_range, 0))

The INDEX & MATCH combination is a powerful alternative to VLOOKUP. It looks for a value in a range and returns a result from another column or row, with more flexibility and better performance.

  • Syntax: =INDEX(return_range, MATCH(lookup_value, lookup_range, 0)) /return_range: The range to return the value from. /lookup_value: The value to search for. /lookup_range: The range to search within. /0: Stands for exact match.
  • Example: =INDEX(B2:B6, MATCH("Keyboard", A2:A6, 0))
    This searches for "Keyboard" in the range A2:A6, and returns the corresponding value from the same row in B2:B6.
    0 ensures an exact match.
